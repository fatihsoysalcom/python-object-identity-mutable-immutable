# Python Object Identity Mutable Immutable

This example demonstrates Python's object model, specifically the difference between mutable and immutable types and how variable assignments create references. It clarifies how changes to mutable objects can be observed through multiple variable names, a common source of confusion for new Python developers.

## Language

`python`

## How to Run

Save the code as `main.py` and run it from your terminal using `python main.py`.
